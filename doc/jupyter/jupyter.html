Jupyter StuffÔÉÅ

This section contains different modules which are specific to the Jupyter interface. Jupyter allows interactive input and nice structured output. Some of the capabilities are present at several modules. It has been paramount to maintain backward compability.

The modeljupytermagic module sets up different magic cell commands which can be handy.
